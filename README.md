# Introduction to NumPy Exercises

This repository contains exercises and solutions for learning NumPy, a fundamental library for numerical computing in Python.

## Contents

- `numpy_exercises.ipynb` - Jupyter notebook with interactive exercises and solutions
- `numpy_exercises.py` - Python script with all exercises and solutions
- `requirements.txt` - Required Python packages

## Learning Objectives

In these exercises, you will learn:

- How to create, reshape, and manipulate arrays using NumPy
- How to access and modify elements in NumPy arrays
- How to apply key NumPy functions to analyze real-world problems

## Setup

1. Install Python 3.8 or higher
2. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Exercises

### Option 1: Jupyter Notebook (Recommended)
Open `numpy_exercises.ipynb` in Jupyter Notebook or JupyterLab for an interactive experience.

### Option 2: Python Script
Run the Python script to see all solutions:
```bash
python numpy_exercises.py
```

## Exercises Overview

1. **Exercise 1**: Finding the day with highest average pollution level
2. **Exercise 2**: Calculating total forest coverage lost
3. **Exercise 3**: Finding water body with worst average quality
4. **Exercise 4**: Finding region with highest average animal sightings
5. **Exercise 5**: Reshaping bird sightings data and calculating weekly totals
6. **Exercise 6**: Transposing climate data and calculating averages

Each exercise includes:
- A real-world problem statement
- Sample data
- Complete solution with explanation
